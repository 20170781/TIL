# 비동기 처리

## 비동기 처리란

자바스크립트에서는 script 태그가 있으면 코드의 실행을 멈추고 script를 불러온 후에 코드를 재실행하는데, 여러가지 목적으로 코드의 실행을 멈추지 않고 script를 불러오는 자바스크립트의 특성



### 간단한 예제

jQuery의 ajax, setTimeout() 등

**콜백 함수로 비동기 처리로 인한 발생하는 문제점들을 해결할 수 있다.**
